下面会讲解怎样将Hexo与github page 联系起来

关联GitHub Page

大概分为以下几步

  • 配置git个人信息
  • 配置Deployment

1、设置Git的user name和email：(如果是第一次的话)

进入 Git bash(右键）

参考上面配置

然后同样右键进入Git GUI

点击最上方的help，再点击show SSH key

创建SSH key

再输入eval "$(ssh-agent -s)"

最后输入ssh-add ~/.ssh/id_rsa

进入github点击自己头像

找到setting进入，再进入SSH key

!

用记事本打开id_rsa.pub（找到他，在用户文件的.ssh里）

复制粘贴到GitHub

测试：ssh -T git@github.com

你将会看到Hi humingx! You’ve successfully authenticated, but GitHub does not provide shell access.

OK，成功！

配置yml

在根目录下找到yml文件

找到#Deployment

按照如下修改

deploy:

type: git

repo: git@github.com:yourname/yourname.github.io.git

branch: master

新建一篇博客，执行下面的命令：

hexo new post "article title"

hexo d -g #部署到github

踩坑提醒

1）注意需要提前安装一个扩展：
npm install hexo-deployer-git --save
1

fatal: Could not read from remote repository.

and the repository exists.

则是因为没有设置好public key所致。
